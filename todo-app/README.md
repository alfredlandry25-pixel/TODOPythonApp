# Flask To-Do List

A small beginner-friendly Flask app that stores tasks in memory.

## Run with Windows Command Prompt

```bat
cd %USERPROFILE%\Desktop\TODO\todo-app
py -3.13 -m venv .venv
.venv\Scripts\activate
python -m pip install -r requirements.txt
python app.py
```

Then open this address in your browser:

```text
http://127.0.0.1:5000
```
