# TaskFlow Todo App

A simple and clean todo app built with React and Vite.

## Features

- Add new tasks
- Mark tasks as complete or active
- Delete tasks
- Filter tasks by All, Active, or Done
- Set task priority as Low, Medium, or High
- Clear completed tasks
- Saves tasks in the browser with localStorage
- Responsive design for desktop and mobile

## Getting Started

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

## Tech Stack

- React
- Vite
- CSS
